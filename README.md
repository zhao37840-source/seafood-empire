# 🦪 海鲜帝国

> AI自动化获客与内容生产系统

## 概述

金源冻品生蚝批发的AI数字化转型项目，通过多个专业化Agent协作，实现：
- 短视频内容自动生产
- 多平台分发与数据追踪
- 精准流量获取与转化

## 技术栈

- **Agent框架**: OpenClaw
- **AI模型**: MiniMax-M2.7
- **视频处理**: FFmpeg, yt-dlp
- **配音**: ElevenLabs (Tingting中文语音)
- **平台**: YouTube, Telegram
